# 🎨 HTML 可视化编辑器 Pro

一款现代化的 HTML 可视化编辑器，支持所见即所得的拖拽设计、实时代码预览、以及无缝的元素编辑体验。

🔗 **在线体验**: [https://a8w5szk4-38082p-devc.preview.with.woa.com/](https://a8w5szk4-38082p-devc.preview.with.woa.com/)

---

## ✨ 功能特性

### 🖱️ 可视化编辑
- **双击编辑**: 在预览区域直接双击任何文本元素即可实时修改内容
- **元素选择**: 点击预览区域中的任意元素，右侧属性面板自动同步显示该元素的所有属性
- **元素树导航**: 左侧面板展示完整的页面 DOM 结构，方便快速定位和选择元素
- **悬停高亮**: 鼠标悬停在预览区的元素上时，会自动高亮显示并显示元素标签

### 📁 文件导入
- **拖拽上传**: 将 HTML 文件拖拽到页面任意位置即可导入编辑
- **文件选择**: 点击顶部"打开本地文件"按钮，通过系统文件选择器导入
- 支持 `.html`、`.htm`、`.txt` 格式

### 🎨 属性编辑
- **内容面板**: 修改文本内容、HTML 源码
- **样式面板**: 调整颜色、字体大小、字重、对齐方式、内外边距、圆角、边框等
- **属性面板**: 修改链接地址、图片地址、ID、Class 等

### 💻 代码视图
- **设计视图**: 纯可视化预览模式
- **拆分视图**: 左侧预览 + 右侧代码实时同步
- **代码视图**: 完整的 HTML 源码编辑，支持语法高亮
- **代码格式化**: 一键格式化 HTML 代码
- **代码复制**: 一键复制完整 HTML 代码到剪贴板

### 📤 导出功能
- 一键导出编辑后的 HTML 文件

## 🚀 快速开始

### 环境要求
- 任意现代浏览器（Chrome、Firefox、Safari、Edge）
- 无需后端服务器，纯前端应用

### 本地运行

#### 方式一：直接打开
直接用浏览器打开 `index.html` 文件即可使用基础功能。

#### 方式二：本地服务器（推荐）
使用任意静态服务器运行，以获得最佳体验：

```bash
# 使用 Python
python3 -m http.server 8080

# 使用 Node.js
npx serve .

# 使用 PHP
php -S localhost:8080
```

然后访问 `http://localhost:8080`。

### Docker 部署

```bash
# 构建镜像
docker build -t html-editor .

# 运行容器
docker run -d -p 8080:80 html-editor
```

访问 `http://localhost:8080`。

## 📂 项目结构

```
html-editor/
├── index.html      # 主页面，包含完整 UI 结构
├── app.js          # 核心逻辑：编辑器交互、文件处理、属性同步
```

**技术栈**: 
- 纯原生 HTML5 + CSS3 + JavaScript
- 无框架依赖，零构建步骤
- 使用 Tailwind CSS (CDN) 进行快速样式开发
- 使用 Highlight.js 进行代码高亮

## 🛠️ 使用指南

### 1. 导入 HTML 文件
- **方式一**: 点击顶部工具栏 📂 **打开本地文件** 按钮
- **方式二**: 将 `.html` 文件直接拖拽到页面任意位置

### 2. 编辑内容
- **双击编辑**: 在预览区域双击任何文本元素直接修改
- **面板编辑**: 在右侧属性面板的"内容"标签页中编辑
- **源码编辑**: 在"代码"视图中直接修改 HTML 源码

### 3. 修改样式
- 选中元素后，在右侧属性面板的"样式"标签页中调整
- 支持修改：文字颜色、背景颜色、字体大小、字重、对齐方式、内外边距、圆角、边框等

### 4. 导出文件
- 点击顶部工具栏的 📥 **导出 HTML** 按钮
- 文件将自动下载到本地
